静态资源

本页面是静态资源展示页面。

在 Django 中，有一个叫做静态资源嗅探器的东西，你可以在 settings.py 中找到一个叫 STATIC_URL 的变量，只要你指定了这个变量的值，那么 Django 框架就会开启静态资源嗅探器。

注意， STATIC_URL 这个变量跟我们实际的存储目录名称没有关系，它的值实际上是给客户端看的。

为了实现这个目的，我在 static_app 中新建了一个 static 目录，并在里面放上了几张人类圣经。

这里有一点需要单独提出来注意一下：

在 settings.py 中的 STATIC_URL 后面指定的值跟静态资源文件的存放路径没有任何关系，它的值被用于渲染的网页中静态资源的 URL 而不是服务器这边的存储路径。

一般建议，这个值能改还是改一改，因为你的服务器架构可能会被恶意攻击者猜到，进而对你的服务器实施攻击。
